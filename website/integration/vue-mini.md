# Vue-mini 集成

## vue-mini 介绍

Vue Mini 是一个基于 Vue 3 的小程序框架，它简单，强大，高性能。

你可以使用 Vue 3 的响应式数据 + 组合式 API 开发小程序。

文档地址: https://vuemini.org/

## 集成

你可以非常简单的在 `weapp-vite` 中集成 `vue-mini`

只需执行 `pnpm i @vue-mini/core` 安装包之后即可直接使用

然后你就可以跟随文档，使用 `vue-mini` 的所有特性了！
